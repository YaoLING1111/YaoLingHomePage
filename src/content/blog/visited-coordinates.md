旅行不是研究主页的主线，但它持续改变我理解工程、城市和生活节奏的方式。

## East Asia

- **China / 中国**：上海交通大学是学习与研究路线的原点。
- **Japan / 日本**：在仙台参加暑期工程项目，用 ROS、YOLOv8 与 A\* 搭建自主小车。
- **Thailand / 泰国**：在高密度学习和项目周期之间留下一段热带坐标。

## Europe

- **Sweden / 瑞典**：在 Stockholm 的 KTH 交换学习，把机器人、估计和北欧冬天放进同一个学期。
- **Finland / 芬兰**：安静、清冷、有秩序，适合写代码和思考模型。
- **France / 法国**：艺术、城市与漫游感，为理工训练补充另一种观察方式。
- **Germany / 德国**：控制、工程与系统感在这里形成自然呼应。
- **Austria / 奥地利**：音乐和山之间的一站。
- **Romania / 罗马尼亚**：不喧哗，却很容易被记住的东欧坐标。
- **Spain / 西班牙**：阳光、广场和更舒展的生活节奏。
- **Portugal / 葡萄牙**：欧洲西侧的边界感，像地图翻页前的一枚句号。
