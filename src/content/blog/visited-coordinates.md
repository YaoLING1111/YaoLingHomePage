旅行不是研究主页的主线，但它持续改变我理解工程、城市和生活节奏的方式。

## East Asia

- **China / 中国**：主页原点。生于此，长于此。
- **Japan / 日本**：仙台的夏。在鲁迅跟随藤野先生学习的学校度过暑研。
- **Thailand / 泰国**：第一次出国。记忆里的榴莲和山竹碰撞出东南亚的夏天。

## Europe

- **Sweden / 瑞典**：斯京的教堂钟声还在回荡。宫崎骏动画里的城市。漫天白雪里我看见自己的灵魂。
- **Finland / 芬兰**：坐船从斯京出发，去赫尔辛基。小小的城市，清冷的气质。
- **France / 法国**：从卢浮宫到巴黎圣母院，从凯旋门到塞纳河游船。太多的名胜。一直想要去的地方，浪漫与艺术是心底的召唤。
- **Germany / 德国**：严肃与荒诞戏谑共存。随处可见的涂鸦，从柏林墙一直挤到旅馆电梯墙上。
- **Austria / 奥地利**：马车缓缓驶过茜茜公主的城堡。维也纳用音乐和艺术，将它的痛苦抚平。
- **Romania / 罗马尼亚**：东欧小城。仿佛回到故乡的乡间。一切都在讲述曾经繁荣。
- **Spain / 西班牙**：阳光、海滩，日出。受够了漫长黑夜的北欧居民，总念叨的度假圣地。建筑学者必须朝圣的地方。海鲜饭也甚是美味。
- **Portugal / 葡萄牙**：带轨的黄色电车和金灿灿的蛋挞。可以去海鲜市场一品美味。
